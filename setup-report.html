設定練習回報系統

三步驟自動收集學生練習紀錄到 Google Sheets

1
建立 Google Sheet
開啟一個新的 Google 試算表，將檔案命名為「學生練習紀錄」。
然後在第一列（Row 1）輸入以下表頭：
時間戳記 姓名 工具 關卡 分數 星數 金幣 驗證碼
📄 點此建立 Google Sheet
2
部署 Apps Script
在剛才建立的 Google Sheet 中，點選上方選單
「擴充功能 → Apps Script」，
將下方的程式碼全部複製，貼到編輯器中（取代原有內容），然後儲存。
function doPost(e) { try { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow([ new Date().toLocaleString('zh-TW', {timeZone: 'Asia/Taipei'}), data.name || '未填寫', data.tool || '', data.level || '', data.score || 0, data.stars || 0, data.coins || 0, data.verifyCode || '' ]); return ContentService .createTextOutput(JSON.stringify({status: 'success'})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({status: 'error', message: err.toString()})) .setMimeType(ContentService.MimeType.JSON); } } function doGet(e) { return ContentService .createTextOutput(JSON.stringify({status: 'ok', message: '回報系統運作中'})) .setMimeType(ContentService.MimeType.JSON); }
儲存後，依照以下步驟部署為網頁應用程式：
  1. 點選右上角 部署 → 新增部署
  2. 左側齒輪圖示選擇類型為 網頁應用程式
  3. 「誰可以存取」改為 所有人
  4. 點 部署，授權後複製產生的網址
3
貼上網址
部署完成後，將 Apps Script Web App 網址貼到下方，
然後測試連接是否正常，最後按儲存設定。

🔧 測試回報

手動發送一筆測試資料到 Google Sheet，確認系統正常運作。
