📚 二、英文教學流程（每堂課 SOP）

🎯 30 天核心策略

  • 不求滿分，搶每一分能拿的分
  • 聽力優先 → 最容易搶分（聽到關鍵字就能選對）
  • 單字只要求「認得」，不要求「拼出來」
  • 不教文法規則，用「口訣＋直覺填空」取代
  • 不做長篇閱讀（挫折感太大）

⏱ 每堂課 40 分鐘 SOP

Phase 1（0-8 分鐘）🧘 老師帶學

  • 大螢幕開 vocab-adventure.html → 當天的 Day → 學習模式
  • 每個字按 🔊 唸 2 遍，讓學生跟著唸
  • 有例句的字，按紫色 🔊 聽整句
  • 帶完問：「哪個字最好笑？最難？」
  • ⚠️ 不要讓他們自學，他們會發呆

Phase 2（8-22 分鐘）📱 手機互動練習

  • 學生掃 QR Code 自己練
  • 先做「🔊 聽力闖關」→ 再做「🎯 配對挑戰」
  • 老師巡視，幫卡住的學生
  • 做完的人可以挑戰「👹 BOSS 戰」
  • 鼓勵：「誰拿到三顆星了？」

Phase 3（22-32 分鐘）📋 紙本迷你學習單

  • 從剛才練過的字＋句型出 3-5 題
  • 格式仿會考（四選一）
  • 當堂寫、當堂對答案
  • 教答題技巧：刪去法、關鍵字定位法

Phase 4（32-40 分鐘）🏆 收尾＋獎勵

  • 學生打開「📊 給老師看」讓你確認進度
  • 唸出全班今天的總星數（正向鼓勵）
  • 剩餘時間：Wordwall 或 PaGamO 當獎勵
  • 預告明天：「明天是 Day __！」
確認學生有做：讓學生打開「📊 給老師看」畫面，上面有完成時間＋驗證碼，沒做完的不會出現紀錄。

🎯 四、會考答題技巧（教學生的）

策略怎麼教
🔍 關鍵字定位法聽到/看到認識的字 → 選有那個字的選項
❌ 刪去法看不懂先不管，把「一定不是」的刪掉
🖼 圖片線索法會考很多題有圖，先看圖再看選項
🔄 相似字排除四個選項裡太像的通常不是答案
🎲 不要留空白猜也要猜，四選一有 25% 機率

📐 五、數學動畫教學指引

內角和系列（建議教學順序）

順序動畫概念關鍵動作
1🔺 三角形內角和 = 180°撕下 3 個角 → 拼成一條直線
2🔷 四邊形內角和 = 360°切 1 刀 → 2 個△ → 2×180°
3⬠ 五邊形內角和 = 540°切 2 刀 → 3 個△ → 3×180°
教完三個，問學生：「你發現規律了嗎？」
→ 邊數 - 2 = 三角形數量 → ×180° = 內角和
→ 六邊形？(6-2)×180° = 720°！

外角和系列

動畫概念關鍵動作
🚗 外角和任何多邊形外角和 = 360°小車跑一圈，外角撕下來拼成完整圓
先用三角形版本教「什麼是外角」（灰色內角 vs 彩色外角），再切換四邊形、五邊形，讓學生自己發現「都是 360°！」

🔧 六、常見狀況處理

狀況怎麼處理
學生不想拿手機讓他跟旁邊的人一起用一支，合作反而更投入
學生一直玩其他 app「先給我看📊學習紀錄，有完成才能下課」
學生說太簡單「那你去打 BOSS 戰拿三顆星給我看」
學生說太難「沒關係，先用學習模式聽過就好，不用背」
學生中途跳出紀錄裡看不到完成時間 = 沒做完，一眼就知道
手機沒有聲音確認手機沒有靜音，或插耳機
網頁打不開確認有網路，用 Chrome 瀏覽器開啟最穩定
